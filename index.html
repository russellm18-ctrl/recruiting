Questions

We don't have recent photos — can you still build it?

Yes. That's exactly what the Profile tier is for — a complete, optimized page built from photos you already have. If you want fresh action and portrait imagery, choose Profile + Shoot.

How fast does it go live?

Profile pages are live within 14 days. Shoot-tier pages are built after your session. All-American includes priority delivery. Need it faster? Add Priority Deployment at checkout.

Will coaches actually find it?

That's the entire point. Every page is built with structured data and SEO so it surfaces when a coach searches the athlete's name — not buried in a DM or a PDF. It's the same technical work I use across my own sites.

Do you only shoot distance and track?

Endurance — track, cross country, and distance — is home base. The profile framework works for any sport, so if you're elsewhere, reach out and we'll talk.

Can we update it later?

Yes. The $129/yr Season Update refreshes marks, film, and schedule each season so the page never goes stale during recruiting. Drop a new PR mid-season? The $50 Spike Update gets it live within 48 hours.
